# 🤝 Contributing to dotref

## No coding required
The most valuable contribution right now is data.
Add a `.toml` file for a subsystem you know well.

## Steps
1. Fork the repo
2. Create or edit a file in `data/<subsystem>/`
3. Follow the format in the README
4. Open a PR with a link to your source

## Want to build the CLI?
Open an issue — let's talk about language and architecture first.
